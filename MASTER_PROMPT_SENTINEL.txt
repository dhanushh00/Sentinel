PROJECT TITLE:
Sentinel: An Agentic Financial Intelligence Platform for Market Analysis and Investment Insights

OBJECTIVE:
Build a production-quality Final Year Project by extending an existing open-source Stock Dashboard (StockBoard) into an AI-powered Agentic Financial Intelligence Platform.

This is NOT a stock prediction project.

The goal is to build an intelligent financial analysis system that uses multiple AI agents to analyze market data, news sentiment, technical indicators, portfolios, and generate explainable investment insights.

The project must be impressive for recruiters, academically defendable, and realistically achievable within a final-year timeline.

=========================================================
PROJECT VISION
=========================================================

Sentinel is an Agentic Financial Intelligence Platform.

Users can:

- Search stocks
- View market insights
- Read AI-generated reports
- Analyze sentiment
- Understand technical indicators
- Compare companies
- Analyze portfolios
- Chat with AI financial assistants

The system must prioritize:

- Explainability
- Transparency
- Modular architecture
- Agent-based workflows

The system MUST NEVER claim guaranteed stock predictions.

Avoid:

"Tomorrow this stock will increase by 10%"

Instead use:

"Based on current sentiment, technical indicators, and news, the outlook appears moderately bullish."

AGENT DESIGN
=========================================================

AGENT 1:
NEWS AGENT

Responsibilities:

- Fetch latest financial news
- Summarize articles
- Extract important events
- Provide concise insights

Example:

"TCS signed a major international cloud agreement."

Output:

{
summary:
importance:
marketImpact:
}

=========================================================

AGENT 2:
SENTIMENT AGENT

Responsibilities:

Analyze:

- News headlines
- News articles

Return:

Positive
Negative
Neutral

Example:

Positive: 68%
Negative: 22%
Neutral: 10%

Use:

- FinBERT
OR
- HuggingFace financial sentiment models

=========================================================

AGENT 3:
TECHNICAL ANALYSIS AGENT

Calculate:

RSI

MACD

Support/Resistance

Trend Detection

Trend:
Uptrend

AGENT 4:
RISK ANALYSIS AGENT

- Confidence level
